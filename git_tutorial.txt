Git Tutorials

Install Git
Create an account
At the command prompt
To set a user Name
$ git config --global user.name "Ganesh Saripalli" 

To set a email id
$ git config --global user.email "ganesh.saripalli@yahoo.com" 


$ git config --list

$ git config user.email
$ git config user.name

$ git help

$ git help commit

Create a new folder

Go to that folder
$ git init

list all the hidden files as well (but don't delete them)
$ ls -la

//Add all the files in the current directory.
$ git add .

//Commit all the files
$ git commit -m "This is my first commit"

// TO view who commit code

$ git log

$ git log --author="Ganesh"

// Status of git
$ git status

// Add a new files
$ git add <filename>.txt

//git workflow
working copy --> staging area --> repository
